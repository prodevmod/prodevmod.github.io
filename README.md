# WarriorOS

This project is a web-based operating system that I created from scratch using JavaScript, CSS, and HTML. I used the guide from Hacklab flavourtown and this is a project I will ship in Hacklab Stardance. 

## More about the Project

It's a medieval-themed OS because why not and I love warriors. It includes three apps. The welcome one that is accessed from the top bar in WarriorOS UI button, the Quest Archive one basically a journal of imaginary battles and imaginary monsters which  was heavily based on D&D and The About Me app that contains a summary of me, a contacts tab, and a portfolio tab.

And the summary of me is a short but pretty good summary of my journey through the software and hardware world while explaining a bit about my passions. 

## UI
As you can see from my poor art, I used pixel art for drawing everything in regards to photos except the background of the OS and the welcome screen GIF. The rest of the UI I made through HTML and CSS. It was really straightforward. I mostly followed the Hacklab guide while adding my touch. 

## The top bar 
In the top bar, as traded previously, there is the WarriorOS UI button that you can press and it opens up the welcome screen. Then there is the "mens et manus," a very inspiring quote I read recently. I chose it because I believe that both practical and theoretical work is needed for everything and I think it's a motto that best describes my life.Last but not least there is a clock with the date and time on it on the right side of the top bar.
